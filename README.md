## Job Portal – MERN Stack Web Application

This project is a full-stack Job Portal web application developed using the MERN stack (MongoDB, Express.js, React.js, Node.js). It provides a platform where job seekers can find jobs and recruiters can post job openings, making the hiring process simple and efficient.

# Project Overview

The Job Portal system connects Job Seekers and Recruiters on a single platform. It automates the recruitment process by allowing users to search, apply, and manage jobs easily.

# Frontend

The frontend is developed using React, providing a responsive and interactive user interface.

# Features:
Login / Signup Page
Job Listings Page
User Dashboard
Recruiter Dashboard

# Functionality:
Component-based architecture
Responsive design using HTML, CSS, JavaScript
Dynamic job display using cards
API integration for real-time data

# Backend

The backend is built using Node.js and Express.js.

# Responsibilities:
User authentication (login/signup)
Job posting and management
Application handling
API development (REST APIs)

# Database

The application uses MongoDB for storing data.

# Collections:
Users → Stores user details
Jobs → Stores job postings
Applications → Stores job applications
# User Roles
# Job Seeker
Search for jobs
Apply for jobs
View applied jobs

# Recruiter
Post job openings
Manage job listings
View applicants

# Workflow

Login → Dashboard → Recruiter Posts Job → Job Stored in Database → Job Seeker Views Jobs → Applies → Data Stored → Recruiter Reviews Applications

# Installation & Setup
1. Clone the Repository
git clone <your-repo-link>
cd job-portal

2. Install Dependencies
npm install

3. Run Frontend
npm start

4. Run Backend
cd backend
npm install
npm run dev
